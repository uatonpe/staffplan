⚙️ Setup Guide

One-time setup

1) Google Sheet tabs तयार करा

StaffMaster, FormQuestions, Entries हे 3 tabs हवेत. README.md मध्ये exact columns दिले आहेत.

2) Apps Script deploy करा

Apps Script मध्ये Code.gs paste करा. Deploy → New deployment → Web app → Anyone with link.

3) API URL इथे paste करा

4) GitHub Hosting

हा index.html GitHub repo मध्ये upload करा आणि Settings → Pages मधून site live करा.
